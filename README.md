# Schraubenkönig Dresden - Landing Page

Eine moderne, professionelle Landing Page für Schraubenkönig Dresden, spezialisiert auf hochwertige Befestigungstechnik.

## Features

- **Modernes Design**: Industrielle Optik mit klarem, professionellem Layout
- **Responsive**: Vollständig responsive für alle Geräte
- **Astro Framework**: Schnell, modern und SEO-optimiert
- **Sektionen**:
  - Hero mit Call-to-Action
  - Produktübersicht
  - Über uns
  - Kontaktformular

## Entwicklung

```bash
# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev

# Für Produktion bauen
npm run build

# Vorschau der Production-Build
npm run preview
```

## Technologie-Stack

- [Astro](https://astro.build/) - Static Site Generator
- Vanilla CSS mit modernen Features
- Keine externe Dependencies für maximale Performance

## Struktur

```
/
├── public/          # Statische Assets
├── src/
│   ├── components/  # Wiederverwendbare Komponenten
│   ├── layouts/     # Layout-Templates
│   └── pages/       # Seiten (index.astro)
└── package.json
```
